# Pregnancy-tracker

Full stack application for showing info about the pregnancy status and a todo list

  <img src="screen.png" />
</p>


# Getting started

  Open the console and run:
  
  - in the server folder:
    ```
    npm install
    ```

    ```
    npm run dev  
    ```

  - in the client folder:
    ```
    npm install
    ```

    ```
    npm run dev
    ```
Open the browser and go to  http://localhost:5173/
