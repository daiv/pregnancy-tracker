# Getting started

  open the console and run:
  
  - in the server folder: 
      1- npm install
      2- nodemon server.js

  - in the client folder: 
      1- npm install
      2- npm run dev

now in the browser go to  http://localhost:5173/

# Tech stack

* Client: 
  * React 
  * Vite

* Server:
  * Express
  * MongoDb
  * Mongoose
  * Nodemon
